## 📖 Overview

CarTrust AI is an advanced vehicle intelligence platform designed to help users evaluate used vehicle listings through AI-powered analysis.

The platform analyzes vehicle specifications, pricing patterns, mileage data, ownership information, and visual vehicle evidence to generate professional inspection reports and trust assessments.

CarTrust AI assists buyers in identifying:

- 🚨 Fraudulent vehicle listings
- 📉 Unrealistic pricing anomalies
- 🔧 Potential maintenance liabilities
- 📊 Market value estimates
- ⚠️ Odometer rollback risks
- 🏷️ Specification inconsistencies
- 📄 Professional AI-generated inspection reports

## ✨ Features

## 🚀 Run Locally

### Prerequisites

- Node.js 18+
- Google Gemini API Key

### Installation

1. Clone the repository

```bash
git clone https://github.com/davedhruvansh/CarTrustAI.git
```

2. Navigate to project directory

```bash
cd CarTrustAI
```

3. Install dependencies

```bash
npm install
```

4. Create `.env.local`

```env
GEMINI_API_KEY=YOUR_API_KEY
```

5. Start development server

```bash
npm run dev
```
